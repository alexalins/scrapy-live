# Live Scrapy

## Instalando
```
pip install scrapy
```

## Iniciando Projeto
```
scrapy startproject MEU_PROJETO
```

## Criando Spider
```
scrapy genspider NOME LINK
```

## Rodando 

```
scrapy runspider CAMINHO/ARQUIVO.py
```

## Exportando (CSV, JSON ou XML)
```
scrapy crawl NOME_SPIDER -o NOME_ARQUIVO.json
```
